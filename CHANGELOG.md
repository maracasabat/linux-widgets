# Історія змін

Формат відповідає [Keep a Changelog](https://keepachangelog.com/uk/1.1.0/),
версіонування — [SemVer](https://semver.org/lang/uk/).

## [1.2.0] — 2026-07-18

### Додано
- Режим **«Віддалено через SSH»**: якщо ти запускаєш Claude на іншій машині
  (наприклад, VS Code Remote-SSH), віджет може показувати її ліміти. Запит
  (читання токена + виклик API) виконується **на віддаленій машині** — токен
  її не залишає, каналом SSH повертається лише JSON зі статистикою
- Налаштування джерела даних: «Локально» / «Віддалено через SSH», поле
  `user@host` та (за потреби) власна віддалена команда
- Розділ у README про безпеку SSH-режиму й приклад захищеного ключа
  (`command=...` у `authorized_keys`)

### Змінено
- Зміна налаштувань вигляду більше не спричиняє зайвий мережевий запит —
  повторний запит робиться лише коли змінюється джерело даних

## [1.1.1] — 2026-07-15

### Виправлено
- Зміна налаштувань (розмір, шрифт, кольори тощо) тепер застосовується
  наживо. Cinnamon доставляє зміни через DBus, але його діалог налаштувань
  іноді губить це сповіщення — значення записувалися у файл, а працюючий
  десклет лишався зі старими до перезавантаження. Тепер десклет сам стежить
  за своїм файлом налаштувань і надійно застосовує зміни (~250 мс)

## [1.1.0] — 2026-07-15

### Додано
- Конфігурація через змінні середовища та файл
  `~/.config/claude-usage-desklet/.env`: `CLAUDE_OAUTH_TOKEN`,
  `CLAUDE_CREDS_PATH`, `CLAUDE_USAGE_URL` (+ `.env.example`)
- Налаштування шляху до `credentials.json` у діалозі налаштувань
- Кастомізація вигляду: розмір шрифту, висота прогрес-барів, заокруглення
  кутів, колір фону
- Кастомізація кольорів: заголовок, текст, кольори індикаторів
  (низьке / попередження / критичне) та їхні пороги
- Перемикачі елементів: заголовок, час останнього оновлення, час скидання
- Розширена документація (README) та історія змін (CHANGELOG)

### Змінено
- Налаштування згруповано на три сторінки: «Загальні», «Вигляд», «Кольори»

## [1.0.0] — 2026-07-15

### Додано
- Перший випуск: десклет Cinnamon, який показує ліміти Claude —
  сесія (5 год), тижневі (всі моделі та по моделі), додаткове використання
- Прогрес-бари з кольоровою індикацією та часом скидання лімітів
- Автооновлення за інтервалом і оновлення кліком по віджету
- Скрипти `install.sh` / `uninstall.sh` (симлінк + вмикання через gsettings)
